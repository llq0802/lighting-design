# 介绍

![GitHub](https://img.shields.io/github/license/doly-dev/antd-more.svg)

Lighting-Design 是基于 Ant Design 扩展的业务场景组件。可以显著的提升制作 CRUD 页面的效率。

## 特性

- 💎 简单易用 - 基于 Ant Design 进行封装
- 📦 场景丰富 - 面向业务场景封装
- 💡 TypeScript - 提供 TypeScript 类型定义
