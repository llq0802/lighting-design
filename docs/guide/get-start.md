## lighting-design

lighting-design 是基于 Ant Design 而开发的业务常用模板组件，提供了更高级别的抽象支持，开箱即用。可以显著的提升制作 CRUD 页面的效率，更加专注于页面。

## 安装

---

```git
npm i lighting-design

#or

yarn add lighting-design

#or

pnpm add lighting-design

```

## 使用

---

```ts
import { LColor } from 'lighting-design';
```
