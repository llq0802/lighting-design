## 介绍

---

rc-use-hooks 是一个使用 React + TS 业务开发中常用的简单易用的 Hooks 库

## 安装

---

```git
npm i rc-use-hooks

#or

yarn add rc-use-hooks

#or

pnpm add rc-use-hooks

```

## 使用

---

```ts
import { useShow } from 'rc-use-hooks';
```
