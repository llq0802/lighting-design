## API

```ts
import { LRightMenu } from 'lighting-design';
```

除了以下参数，其余和 antd [Dropdown](https://ant-design.antgroup.com/components/dropdown-cn#api) 组件一样。

| 参数     | 说明               | 类型        | 默认值 |
| -------- | ------------------ | ----------- | ------ |
| children | 需要右键菜单的容器 | `ReactNode` | `-`    |
